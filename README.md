家庭金融管理系统设计文档
=====================
# 一.需求分析
该系统提供家庭金融管理功能，旨在为家庭提供一个便捷的记账和理财收入记录工具，假如你是一位家庭主妇或者家庭的金钱的管家，这款应用将大大增加你记账的效率并且为你提供以月年为周期的账目统计。
 ## 功能介绍：
（1）支持录入，修改，删除和查询家庭成员和所有家庭成员的角色。  
（2）支持录入，修改，删除和查询每天各项的收入和支出。  
（3）支持录入，修改，删除和查询固定收入。  
（4）支持月度，季度，年度等收支统计。  
（5）支持录入，修改删除和查询各项资金的分配使用。  
（6）其他需要的功能后续添加。
